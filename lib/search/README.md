# Search Service

Camada planejada para busca full-text, busca vetorial, ranking híbrido e reranking opcional.

Entradas típicas:

- query textual
- filtros de ano, autor, tópico, tipo e status
- modo de busca
- paginação

Saída esperada:

- resultados ranqueados
- score por estratégia
- metadados necessários para UI e RAG
